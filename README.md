# A YouTube API Comment bot, better faster and free!


# Requirements:

  - Python 3+
  - A YouTube Channel
  - A YouTube API (I show how to get one in the video, it's really easy)


# Modules:
  - google-api-python-client
  - google-auth
  - google-auth-oauthlib
  - google-auth-httplib2
  - requests
  - bs4
  - random
